# Sample Data

This directory contains **demo/sample data only**. It is not a recommendation of any trading instrument.

- `ohlcv.csv` uses the placeholder symbol `DEMO-SYMBOL` so backtest and validation examples run without implying a default instrument.
- Replace this data with your own historical CSV (with columns: `date`, `symbol`, `open`, `high`, `low`, `close`, `volume`) for real analysis.
- Atlas does not choose a trading symbol for you. Configure your own symbol/watchlist based on your broker/API provider and supported paper/live asset classes.
