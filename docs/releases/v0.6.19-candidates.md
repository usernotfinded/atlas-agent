## Implemented

- **CAND-011:** Kill-Switch `last_heartbeat()` Type-Safety Cleanup.
  - Narrows the audit payload construction in
    `src/atlas_agent/safety/kill_switch.py` to eliminate the pre-existing mypy
    `union-attr` warning on `last_heartbeat().isoformat()`.
  - Replaces the repeated `self.heartbeat_manager.last_heartbeat()` conditional
    expression with a single local variable and an explicit `is not None` check
    before calling `.isoformat()`.
  - Preserves the `last_heartbeat` audit payload key and the `str | None` value
    shape.
  - Adds mandatory `heartbeat_expired` audit-payload regression tests in
    `tests/safety/test_kill_switch_v2.py` covering both the ISO-string branch
    and the `None` branch.
  - Does not change kill-switch decision logic, fail-closed behavior, return
    values, heartbeat/deadman behavior, `RiskManager`, audit hash-chain, or any
    broker/provider/order/credential/network behavior.
