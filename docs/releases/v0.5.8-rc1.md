`v0.5.8rc1` is a **release candidate**, not a stable final release. It exists so reviewers and maintainers can verify the v0.5.8 feature set, docs, and safety posture before a stable tag is created.

## Safety posture

- **Not financial advice.** Atlas Agent does not imply profitability or trading correctness.
- **Not proof of trading safety.** Risk gates reduce unintended actions but do not eliminate trading risk.
- **Not proof of profitability.** No backtest, simulation, or paper result guarantees real-money performance.
- **Not production trading ready.** Live trading is disabled by default and requires explicit multi-gate opt-in.
- **Live trading** remains disabled by default.
- **Provider execution** remains locked (governed by risk manager and artifact policy, not a runtime API block).
- **Broker execution** remains blocked unless explicit opt-in gates pass.
