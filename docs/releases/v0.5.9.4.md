# Atlas Agent v0.5.9.4

Public trust status: [v0.5.9.4 Trust and Release Status](../trust/v0.5.9.4-status.md).

## Summary

Atlas Agent `v0.5.9.4` promotes the already-implemented `0.5.9.4` package/source version to a public GitHub release so update detection can see the released version.

This release is a workflow, documentation, and release-identity cutover. It does not change runtime trading behavior.

## Included Changes

- Source/package version remains `0.5.9.4`.
- Public GitHub release identity is `v0.5.9.4`.
- Release docs, trust center status, release readiness notes, and release identity checkers were aligned with `v0.5.9.4`.
- Main health and artifact hygiene context from prior maintenance updates remains in place.

## CI / Workflow Maintenance

- GitHub Actions were modernized to Node 24-compatible action majors.
- `actions/checkout` was updated to `@v6`.
- `actions/setup-python` was updated to `@v6`.
- `actions/upload-artifact` was updated to `@v6` where artifact upload is used.
- A GitHub Actions version guard was added to prevent regression to deprecated action majors.

## Safety Defaults

- No runtime trading behavior changes.
- No live trading default changes.
- No provider execution default changes.
- No broker execution default changes.
- Live trading remains disabled by default.
- Provider execution remains disabled by default.
- Broker execution remains disabled by default.
- Human approval and deterministic risk gates remain required for live order flow.
- Autonomous trading is not claimed.
- This software is not financial advice.
