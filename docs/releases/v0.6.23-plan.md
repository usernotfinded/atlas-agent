## Safety boundaries

This release enables nothing at runtime. Specifically, no candidate or document
in `v0.6.23`:

- enables live trading or makes live trading a default;
- enables live order submission or changes `can_submit`;
- enables broker or provider execution, or calls real brokers or providers;
- loads broker/provider credentials;
- adds network access;
- places, cancels, or flattens orders, or creates pending orders;
- mutates approval queues, the kill switch, heartbeat, deadman, or the audit
  hash-chain;
- broadens the CAND-014 provider-artifact extraction boundary.

`atlas run --mode live` remains fail-closed. This is a GitHub-only project and
PyPI remains unpublished.
