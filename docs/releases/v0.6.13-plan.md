## Safety Boundaries

Unless explicitly approved in a future candidate, these boundaries remain unchanged for `v0.6.13`:

- **Live trading** disabled by default.
- **Live submit** disabled by default.
- **Provider execution** disabled by default.
- **Broker order submission** blocked for non-paper brokers.
- **No credentials, secrets, or API keys** committed.
- **No claims that live trading is safe, that risk is absent, that profit is guaranteed, or that production/autonomous trading readiness exists.**
