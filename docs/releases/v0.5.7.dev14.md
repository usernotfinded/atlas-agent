# Atlas Agent v0.5.7.dev14

## Summary

- This is a development tag.
- Focus: local research dossier artifacts and configless dossier dispatch.
- It is not production live-trading readiness.
- No financial outcome is promised.

## Highlights

- `atlas research dossier RUN_ID`
- Local dossier artifacts.
- Deterministic-dossier provider.
- Dossier summary of the local paper-only artifact chain.
- check-artifacts integration for dossiers.
- Timeline integration for run -> dossier lineage.
- Demo integration for dossier creation and post-dossier timeline validation.
- Configless/no-secret dossier dispatch.
- No LLM/API/network behavior is enabled.

## Dossier Workflow

- `atlas research dossier RUN_ID`
- Provider: `deterministic-dossier`
- Creates local dossier artifact.
- Consolidates local paper-only artifact state.
- Does not call LLMs/APIs/network.
- Does not read API keys.
- Does not load `.env.atlas`.
- Does not call brokers.
- Does not create orders/approvals/pending orders.
- Does not authorize live trading.

Path:

```
.atlas/research/<SYMBOL>/dossiers/<dossier_id>.json
```
