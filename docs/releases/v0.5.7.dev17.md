# Atlas Agent v0.5.7.dev17

## Summary

- This is a development tag.
- Focus: LLM Sandbox Contract & Replay System (Batch 7.1).
- It is not production live-trading readiness.
- No financial outcome is promised.

## Highlights

- `src/atlas_agent/research/sandbox_contracts.py` defines deterministic canonical JSON hashing, artifact validation, and safe lineage/symbol validators.
- `atlas research sandbox-list`, `sandbox-show`, `sandbox-validate`, `sandbox-replay`, and `import-provider-response` are now configless research subcommands.
- `atlas research import-provider-response SANDBOX_ID --file FIXTURE.json` validates and imports local provider response fixtures with full contract checks.
- Sandbox request artifacts now include `artifact_type`, `content_hash` (SHA-256), and `contract_version` for deterministic replay.
- `sandbox-replay` recomputes the artifact hash and verifies it matches the stored `content_hash`.
- Timeline lineage now links `run_id -> prompt -> imported provider response` for full auditability.
- All new commands remain configless: no `AtlasConfig.from_env`, no `.env.atlas` secrets, no network calls.

## Safety and Compatibility Notes

- Frozen research scope is paper-only and analysis-only.
- Frozen research commands are local artifact workflows.
- Frozen local research commands do not load `.env.atlas` or config secrets.
- Sandbox commands do not call LLMs, APIs, or network.
- Sandbox commands do not read API keys.
- Sandbox commands do not submit orders or create approvals/pending orders.
- Sandbox commands do not authorize live trading.
- Research commands are not trading advice.
- Evaluation does not produce trading signals.
- Dossier is not a trading decision engine.
- No LLM provider is enabled.
- Simulated provider is not a real external provider.
- Response review is not a real LLM evaluator.
- Live submit remains disabled by default and gated.
- Reconcile remains read-only.
- Safe quote gate remains execution-time only.
