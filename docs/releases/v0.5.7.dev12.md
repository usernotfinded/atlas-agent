# Atlas Agent v0.5.7.dev12

## Summary

- This is a development tag.
- Focus: local prompt packets and simulated provider response artifacts.
- It is not production live-trading readiness.
- No financial outcome is promised.

## Highlights

- `atlas research prompt RUN_ID`
- `atlas research simulate-provider PROMPT_PACKET_ID`
- Prompt packet artifacts for bounded, sanitized future-provider context.
- Deterministic-mock provider response artifacts.
- Redaction/truncation safety for prompt and provider response content.
- Provider response safety checks with deterministic validations.
- `check-artifacts`, `timeline`, and demo integration for prompt and provider-response artifacts.
- No LLM/API/network behavior is enabled.

## Prompt Packet Workflow

- Prompt packets are local JSON artifacts.
- Prompt packets prepare bounded context for future provider work.
- Prompt packets do not call LLMs.
- Prompt packets do not call APIs/network.
- Prompt packets do not read API keys.
- Prompt packets do not submit orders or create approvals/pending orders.
- Prompt packet content is bounded and redacted.

Path:

```
.atlas/research/<SYMBOL>/prompts/<prompt_packet_id>.json
```

## Simulated Provider Response Workflow

- `atlas research simulate-provider PROMPT_PACKET_ID`
- Provider: `deterministic-mock`
- Provider status: `simulated`
- Creates local provider response artifact.
- Validates response safety.
- Does not call LLMs/APIs/network.
- Does not read API keys.
- Does not call brokers.
- Does not create orders/approvals/pending orders.
- Does not authorize live trading.

Path:

```
.atlas/research/<SYMBOL>/provider_responses/<provider_response_id>.json
```
