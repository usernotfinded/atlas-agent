# Atlas Agent v0.5.7-rc4

> **Not financial advice.** Atlas Agent is a software tool, not a financial advisor. Trading involves significant risk of loss.

## Overview

This is the fourth release candidate for the v0.5.7 line.

`v0.5.7-rc4` is a **sandbox/paper/preflight release candidate**. It is not a live-trading release. It does not enable provider execution, broker order submission, or live trading.

## Scope

- CI release gate parity: aligned GitHub Actions with the local RC verification stack.
- Restructured CI into fast PR gate and manual heavy release gate.
- Added clean install verification and package distribution dry-run to CI quick gate.
- Added public docs consistency, README quickstart verification, and RC cutover check to CI.
- Added `docs/ci-release-gates.md` documenting CI structure and safety rules.
- Package version bumped to `0.5.7rc4` (package) / `v0.5.7-rc4` (public tag).
- No runtime behavior changes in this batch.
- No broker/execution/risk/config/safety boundary changes.

## What this is not

- Not a live-trading ready release.
- Not a production trading milestone.
- Not a provider execution unlock.
- Not a broker integration milestone.
- Not a trust grant.
- Not a profitability or trading-correctness claim.
- Not financial advice.

## Safety posture

- Live trading disabled by default.
- Provider execution remains locked.
- Trust remains blocked.
- No broker/order path in the CI workflow.
- No credentials loaded by the CI workflow.
- No network enabled by Atlas runtime in CI.
- No publish, upload, tag, or push performed by CI.
