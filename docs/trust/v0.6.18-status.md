## Scope

`v0.6.18` contains CAND-010, a safety-state persistence regression guard. It
adds a static checker that scans `heartbeat.py`, `deadman.py`, `kill_switch.py`,
and `state.py` for reintroductions of fixed `<target>.tmp` write patterns, and
allows `atomic_write.py` as the unique `mkstemp`-based helper. The release also
fixes a release-assurance protected-boundary remediation brace-escape bug. No
protected runtime safety boundary changed during the release cutover.

## Included Security Hardening

- Safety-state persistence regression guard prevents reintroduction of fixed
  temporary filenames in the four CAND-009 safety modules.
- All existing deterministic risk, approval, kill-switch, dead-man heartbeat,
  and audit hash-chain requirements remain in place.
- No protected runtime safety boundary changed during the release cutover.
