## Safety posture for v0.6

- **Live trading** is disabled by default.
- **Live submit** is disabled by default.
- **Provider execution** remains disabled by default.
- **Broker execution** remains blocked by default unless explicit opt-in gates pass.
- **Reflections, skills, and learning suggestions** are offline, research-only, and require manual review.
- **Dashboard** remains read-only with no trading controls.
- **Notifications** remain disabled/dry-run by default.
- **Fail-closed broker guards** block all unsupported broker paths.

## Deferred to post-v0.6.0

- Real LLM-driven reflection generation, skill mining, and learning nudges.
- Additional broker adapters (Binance completion, CCXT enablement, IBKR implementation).
- Network benchmark data fetch (currently requires local CSV).
- PyPI publish.
- Multi-user cloud deployment.
- Advanced frontend framework dashboard.
