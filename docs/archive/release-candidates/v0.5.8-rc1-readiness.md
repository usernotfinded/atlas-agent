# v0.5.8 RC1 Readiness Dry Run

> **Not financial advice.** Atlas Agent is a software tool, not a financial advisor. Trading involves significant risk of loss.

## What this document is

This is a **readiness dry run** for a `v0.5.8` stable cutover. It exists so maintainers can verify, at any point, whether `main` is in a state that could be considered for an stable cutover.

**This is not a release.** No tag has been created. No package has been published. The current package version is `0.5.8`.

## What RC1 readiness means

- All `must_fix` items from the [v0.5.8 Gap Prioritization Plan](../legacy-plans/v0.5.8-gap-prioritization.md) are closed.
- At least one `should_fix` batch has been evaluated and any accepted items are implemented.
- The full test suite passes.
- The release quick gate passes.
- Public docs do not overclaim readiness, profitability, or safety.
- Protected boundaries are clean.
- No generated artifacts are staged.

## What it does not mean

- **Not a commitment to release.** An RC tag is only created after explicit maintainer review.
- **Not a performance claim.** Readiness checks verify structure, docs, and safety posture, not trading performance.
- **Not a live-trading endorsement.** Live trading remains disabled by default.
- **Not a security audit.** This is a static local check, not a penetration test or formal security review.

## Safety posture

- **Not financial advice.** Atlas Agent does not imply profitability or trading correctness.
- **Not proof of trading safety.** Risk gates reduce unintended actions but do not eliminate trading risk.
- **Not proof of profitability.** No backtest, simulation, or paper result guarantees real-money performance.
- **Not production trading ready.** Live trading is disabled by default and requires explicit multi-gate opt-in.
- **Live trading** remains disabled by default.
- **Provider execution** remains locked (governed by risk manager and artifact policy, not a runtime API block).
- **Broker execution** remains blocked unless explicit opt-in gates pass.
